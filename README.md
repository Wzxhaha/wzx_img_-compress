# wzx_img_compress
A bash script used to compress images

## how to use
1. `cd xxx/wzx_img_compress`
2. create a file folder.For example: img . And put the images(.png，and images name can't have space) in this file folder.
3. `bash WZXShell.sh img`
4. you will get a a file folder that contain .png、@2x.png、@3x.png and Contents.json
